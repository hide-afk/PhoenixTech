Redes - FAQ

1. No se conecta a internet
1. Problemas de conexión Wi-Fi:
• Verifica la conexión Wi-Fi, reinicia el router y asegúrate que está bien conectado el cable de red.

2. Controladores de red desactualizados o corruptos:
• Actualiza los controladores de la tarjeta de red a través del Administrador de dispositivos.

3. Configuración de TCP/IP incorrecta:
• Ejecuta "ipconfig /renew" en la línea de comandos para renovar la configuración de red.

4. Firewall o software de seguridad bloqueando la conexión:
• Revisa la configuración del firewall para asegurarte de que no esté bloqueando la conexión a Internet.
2. No se conecta al servidor
• Confirma si tanto el servidor con la maquina host están conectados en la misma red.
• Asegúrate de que el funcionamiento del servidor es correcto y que es accesible desde otros ordenadores.
• Verifica la conexión haciendo un ping.
• Revisar la configuración de red de ambos dispositivos (máscara de red, Puerta de enlace, DNS, etc.)
3. En Windows puedo navegar por IP, pero no por URL
• Esto se debe a un problema con el DNS.
1. Dirígete al Panel de control, Redes e Internet, Ver el estado y las tareas de red, Cambiar configuración del adaptador. Ahí encontrarás los adaptadores de red de tu equipo.
2. Encontrarás el adaptador Wi-fi y el adaptador Ethernet. Haz clic derecho al que estés conectado actualmente, dirígete a Propiedades, selecciona Protocolo de Internet versión 4, y presiona Propiedades.
3. Asegúrate de que esté habilitada la opción de obtener la dirección del servidor DNS automáticamente.

• Ejecuta el comando ipconfig /flushdns en la línea de comandos para limpiar la caché DNS y actualizar la resolución de nombres.
• Si el problema persiste, reiniciar el router o el servidor DNS (en caso de tener uno) puede resolver el problema.
4. Hay conexión en la red LAN, pero no a internet (WAN)
• Para ver que aplicaciones o archivos están ocupando cierto espacio, dirígete a Configuración, Sistema, Almacenamiento: Aquí podrás ver que es lo que está afectando a tu espacio, solo tienes que elegir qué archivos o aplicaciones borrar.
5. ¿Creo que la velocidad de mi internet está limitada, como puedo comprobarlo?
• Para revisar que tu equipo no esté limitando la velocidad de red, dirígete a Administrador de dispositivos y Adaptadores de red.
• Ahí encontrarás principalmente Wi-fi y GbE, entre otros. Haz clic derecho al que estes usando y dirígete a propiedades, opciones avanzadas.
• Fíjate que en las distintas propiedades estén fijados los valores más altos.
6. No me puedo asociar a un punto de acceso (AP)
• Confirma que el dispositivo desde el que intentas conectarte al punto de acceso Wi-Fi está dentro del rango de IPs de la misma red.
• Verifica si otros dispositivos pueden conectarse al punto de acceso Wi-Fi sin problemas.
• Comprueba si el dispositivo tiene la tarjeta de red Wi-Fi activada y correctamente configurada para buscar y conectarse a redes disponibles.
• Reiniciar el dispositivo desde el que estás intentando conectarte al punto de acceso Wi-Fi para refrescar la configuración de red.
• Verificar si hay actualizaciones de firmware disponibles para el punto de acceso Wi-Fi y aplicarlas si es necesario.
• Si el problema persiste, contacta con tu compañía de red o con el fabricante del punto de acceso.
7. No puedo acceder al router
• Si el router está encendido y estás conectado a este, y aun así no puedes visualizarlo en la red (hacer ping) debe ser porque no estas en la misma red, comprueba tu IP ejecutando el comando ipconfig en el símbolo del sistema.

• Si tu IP no pertenece a la misma red que tu router:
1. Dirígete al Panel de control, Redes e Internet, Ver el estado y las tareas de red, Cambiar configuración del adaptador. Ahí encontrarás los adaptadores de red de tu equipo.
2. Encontrarás el adaptador Wi-fi y el adaptador Ethernet. Haz clic derecho al que estés conectado actualmente, dirígete a Propiedades, selecciona Protocolo de Internet versión 4, y presiona Propiedades.
3. Asegúrate de que esté habilitada la opción de Obtener una dirección IP automáticamente.

• Si aún así no puedes hacer ping a tu router y estás por ethernet, verifica la conexión física o cambia el cable si es necesario.
• También puede ser que el Firewall este bloqueando el ping o el mismo router está ignorándolo como medida de seguridad.
8. No se puede acceder remotamente a un dispositivo de red
1. Verificación de Conexión y Configuración:
• Asegúrate de que el dispositivo esté correctamente conectado a la red y revisa las configuraciones de red, verificando la compatibilidad de IP y subred.

2. Configuraciones de Seguridad y Cortafuegos:
• Revisa y ajusta las configuraciones de seguridad, incluyendo cortafuegos y programas antivirus. Considera abrir puertos en el router, manteniendo prácticas seguras y limitando el acceso a direcciones IP necesarias.

3. Soporte Técnico y Respaldo del Fabricante:
• Si los problemas persisten, comunícate con el soporte técnico del fabricante para obtener asistencia especializada y resolver cualquier obstáculo en el acceso remoto al dispositivo de red.
